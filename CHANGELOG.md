# Changelog

## [LoneWriter v1.3-oraculo (beta)] - 2026-04-01
### Added
- IA del oráculo: nueva funcionalidad de inteligencia artificial para generación de contenido oracular
- Sistema de debate: mecanismo para debatir y refinar ideas entre diferentes perspectivas IA
- Coherencia del compendio: mejora en la consistencia y organización del compendio de conocimiento
- Nuevos servicios: compendiumSearch.js y entityDetector.js para mejor manejo de información
- Actualizaciones en componentes: AIPanel, RichEditor, SettingsModal, Editor y Compendium
- Mejoras en contexto de IA y servicios de IA
- Actualizaciones en estilos CSS para varios componentes
